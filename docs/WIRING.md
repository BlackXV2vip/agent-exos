# 🔌 ربط api.php بالأداة (Exos Agent)

هذا الملف يوثّق العقد بين أداة **Exos Agent** وبين `api.php` على موقعك —
الربط القائم فعليًا والمُختبَر e2e (10/10 مهام، 27 استدعاء أداة).

## البنية

```
┌─────────────┐  ai-sdk v6 (نفس العملية)   POST {"prompt": "…"}   ┌──────────────┐
│ Exos Agent  │ ───────────────────────────────────────────────▶  │   api.php    │
│  (المحرك)   │ ◀───────────────────────────────────────────────  │  (موقعك)     │
└─────────────┘   tool-calls V3          {"ok": true, "content"}  └──────────────┘
                     ▲
        tools/exos-provider/index.ts — مزوّد أصيل داخل المحرك عبر "npm": "file://…"
        (بدون جسر محلي وبدون @ai-sdk/openai-compatible)
```

## التفعيل (أمر واحد)

```bash
bash tools/wire-exos.sh
# ثم: exos-agent run "أنشئ ملف test.txt فيه hello"
```

## عقد api.php (حدّ أدنى)

```
POST <EXOS_SITE>            (header اختياري للقناة المرخّصة: X-Auth-Token: <EXOS_TOKEN>)
Body:    {"prompt": "<النص الكامل المفلطح للمحادثة + تعليمات الأدوات>"}
Resp:    {"ok": true,  "content": "<الرد النصي أو توجيه أداة>"}
         {"ok": false, "error": "<سبب>"}
```

## طبقة الحماية (تُعالَج تلقائيًا)

المزوّد يتعامل مع الحالتين بدون إعداد إضافي:
- **رد JSON نظيف + توكن** ← يمر مباشرة (قناة مرخّصة إن ضبطت `EXOS_TOKEN`).
- **تحدي JS (testcookie)** ← يحلّه تلقائيًا (`toNumbers` + AES-CBC + كوكي `__test`)، نفس سلوك العميل المرجعي.

## الأدوات العشر

بسبب أن api.php يرد نصًا فقط، المزوّد يحقن تعليمات أدوات في البرومبت. لو النموذج
احتاج أداة يرد بالسطر التالي **فقط**:

```
<<EXOS_TOOL:{"name":"bash","arguments":{"command":"…"}}>>
```

فيحوّلها المزوّد إلى `tool-call` حقيقي بمعيار ai-sdk V3 (generate وstream معًا)،
المحرك ينفّذها محليًا، وتعود النتيجة له عبر `Tool result (call …): …` في الجولة التالية.
لو لم يرد توجيهًا، يُمرَّر النص كإجابة نهائية عاديًا — بلا أي كراش.

## متغيرات البيئة

| متغير | الافتراضي | دوره |
|---|---|---|
| `EXOS_SITE` | `https://wormgpte.xo.je/agent_exos_api.php` | رابط api.php |
| `EXOS_TOKEN` | فارغ | توكن القناة المرخّصة (اختياري) |
| `EXOS_TOKEN_HEADER` | `X-Auth-Token` | اسم هيدر التوكن |
| `EXOS_PROVIDER_DEBUG` | `0` | سجلات المزوّد على stderr |
